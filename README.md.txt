# 🥗 Website Nutricionista – Dra. Carolina Mendes

Este é um projeto fictício de um site institucional para uma nutricionista.  
O objetivo principal foi resolver um problema comum enfrentado por profissionais autônomos: a falta de presença digital.  
Antes, a profissional não era facilmente encontrada no Google. Com este site, pacientes em potencial podem conhecê-la e entrar em contato de forma rápida e prática.

---

## 🎯 Objetivo do Projeto

- Criar uma presença digital profissional e atrativa.  
- Aumentar a visibilidade e facilitar o agendamento de consultas.  
- Treinar habilidades de front-end com HTML, CSS e organização de conteúdo.

---

## 📚 Sobre o Estudo

Este projeto foi desenvolvido como parte prática dos meus estudos em desenvolvimento front-end. Durante o processo, pratiquei:

- Estruturação semântica com HTML5  
- Layouts modernos com Grid Layout e Flexbox  
- Responsividade para dispositivos móveis  
- Organização e boas práticas de código  
- Design acessível e limpo, voltado para profissionais da área da saúde

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- Media Queries  
- Flexbox e CSS Grid
